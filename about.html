About STAH Game

STAH Game is a game developed by Algorithm and Programming Laboratory, Department of Informatics, Faculty of Information Technology, Institut Teknologi Sepuluh Nopember.
This game is a modified version of a well-known game, NIM Game. To understand the mathematical model of strategic situations of the game, we did use game theory approach. But, to solve the game and determine the next step, we used dynamic programming approach.
Why dynamic programming? Because it was the easiest and fastest implementation of this game model. Some said learning algorithm was no use, some said it was useful. But, whatever they said, learning something new is fun. Moreover, without beautiful graphics, without complex gameplay, and without sophisticated game maker or game engine, we did create a simple but challenging game by only using game theory and dynamic programming approach :)
